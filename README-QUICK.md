# SISPEC - Sistema Inteligente de Pecuária de Confinamento

## Quick Start

### Windows
Clique duas vezes em `start.bat` e escolha opção 1

### Linux/macOS
```bash
./start.sh
```
Escolha opção 1 para rodar ambos
